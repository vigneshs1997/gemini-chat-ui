
### gemini-chat-ui
UI frontend for question and answer

## React + Vite
This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

## Commands
# 1.To create Vite application
   npm create vite@latest
# 2.To move to created project folder
   cd gemini-chat-ui
# 3.To install node module
   npm install
# 4.To run react vite application
   npm run dev
# 5.To install Bootstrap dependency
   npm i bootstrap@5.3.8

## Git Commands
